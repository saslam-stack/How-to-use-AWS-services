How to use AWS Services

Amazon Web Services (AWS) is the world's most comprehensive and broadly adopted cloud that includes infrastructure as a service (IaaS) and platform as a service (PaaS) offerings. Gaining familiarity with core concepts of cloud computing and the AWS Cloud will help give you confidence as you begin your cloud journey.

Creating website on S3:

Hosting a website on Amazon S3 (Simple Storage Service) has many benefits such as

  • Scalability: S3 can handle thousands of requests per second without any changes to the architecture.
  • Cost-effectiveness: S3 is pay-as-you-use, with no upfront costs or long-term commitments.
  • Security: S3 encrypts all data by default, and you can use tools like AWS Identity and Access Management (IAM) to control access.
  • Global content delivery: When used with CloudFront, S3 can deliver content globally through a network of edge locations. This reduces latency and improves performance for visitors worldwide.
  • Custom domain names: When used with Route 53, S3 allows you to use custom domain names for your website.
  • Analytics and monitoring: CloudFront provides access logs and real-time metrics to help you understand your website's traffic and performance.
  • No server maintenance: You don't need to worry about server maintenance or hosting costs.

You can read AWS documentation here Use Amazon S3 website hosting.
